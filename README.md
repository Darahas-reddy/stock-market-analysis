# 📈 Stock Market Analysis & Prediction

## 📌 Overview

This project performs **end-to-end stock market analysis** using Python.
It includes data collection, cleaning, visualization, and basic machine learning to generate insights from historical stock data.

---

## 📊 Project Structure

```
stock-market-analysis/
│
├── notebooks/

│   ├── 02_visualization.ipynb
```

## 🛠️ Tools & Technologies

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* yFinance API
* Scikit-learn
* mplfinance

---

## 📈 Key Features

* Multi-stock analysis (AAPL, TSLA, GOOGL)
* Data cleaning and preprocessing
* Moving Average (50 & 200 days) analysis
* Daily returns and volatility analysis
* Correlation heatmap between stocks
* Candlestick chart visualization
* Basic machine learning model for stock prediction
